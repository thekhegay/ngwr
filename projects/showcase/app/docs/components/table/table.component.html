API

Property Description Type Default
columns Table columns configuration WrTableColumns Required
items Data items to display any[] | undefined Required
total Total number of items number 1
currentPage Current page number number 1
pageSize Items per page number 15
pageSizeOptions Available page sizes number[] [15, 25, 50]
loading Loading state boolean false
ordering Current sorting state WrTableOrderItem[] []
currentPageChange Emitted when current page changes EventEmitter<number> -
pageSizeChange Emitted when page size changes EventEmitter<number> -
