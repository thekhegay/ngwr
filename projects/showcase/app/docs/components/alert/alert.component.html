How to install

Import WrAlertModule into a component where you want to use.

Basic usage

Component can be used as wr-alert component

Types

Alert is available in 4 types. Type can be set with type attribute

Closeable

Alert can be closed when closeable attribute is set

API

Property Description Type Required Default
[title] set title of alert string yes
[message] set message of alert string | null null
[closeable] make alert closeable boolean false
[type] set type of alert 'info' | 'success' | 'warning' | 'danger' 'info'
