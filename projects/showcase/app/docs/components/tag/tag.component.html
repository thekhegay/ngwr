How to install

Import WrTagModule into a component where you want to use.

Basic usage

Tag can be used as wr-tag component

Tag Component

Color

Tag is available in different colors. Color can be set with color attribute.

@for (color of colors; track $index) { {{ color.charAt(0).toUpperCase() + color.slice(1) }} }

Outlined

If outlined attribute is set, tag will be displayed with stroke and without background.

@for (color of colors; track $index) { {{ color.charAt(0).toUpperCase() + color.slice(1) }} }

Rounded

In need of round shaped tag, you can set rounded attribute

@for (color of colors; track $index) { {{ color.charAt(0).toUpperCase() + color.slice(1) }} }

Transparent

You can set transparent attribute to make tag transparent.

@for (color of colors; track $index) { {{ color.charAt(0).toUpperCase() + color.slice(1) }} } @for (color of colors; track $index) { {{ color.charAt(0).toUpperCase() + color.slice(1) }} }

Hoverable

If you need to hover effect for tag, you can set hoverable attribute.

@for (color of colors; track $index) { Hoverable Hoverable Hoverable Hoverable }

Using with icons

Tag component can easily be enhanced by adding a icon. All you need is set icon name within icon attribute. Also you can set icon position with iconPosition attribute.

Loading state

If you need a loading version of tag, you can easily set loading attribute. You don't need to remove the inner text or icon, button component will maintain its original size.

API

Property Description Type Required Default
[color] set color of tag @for (color of colors; track $index) { '{{ color.charAt(0) + color.slice(1) }}' | } null 'primary'
[outlined] set outlined style for tag boolean false
[rounded] set rounded style for tag boolean false
[transparent] set transparent style for tag boolean false
[hoverable] set hoverable style for tag boolean false
[icon] add icon to tag wrIconName null
[iconPosition] set position of icon 'start' | 'end' 'start'
[loading] set loading state for tag boolean false
