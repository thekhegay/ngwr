Basic usage

Button can be used as wr-btn component or wr-btn directive

Color

Button is available in different colors. Color can be set with color attribute.

@for (color of colors; track $index) { {{ color.charAt(0).toUpperCase() + color.slice(1) }} }

Outlined

If outlined attribute is set, button will be displayed with stroke and without background.

Rounded

Small size

If you need a small sized button, you can set size="small" attribute.

Using with icons

Button component can easily be enhanced by adding a icon. All you need is set icon name within icon attribute. Also you can set icon position with iconPosition attribute.

Disabled state

You can add disabled attribute to prevent interacting with the button.

Disabled Disabled Disabled

Loading state

If you need a loading version of button, you can easily set loading attribute. You don't need to remove the inner text or icon, button component will maintain its original size.

Loading You can use boolean or string Loading with icon

Disabled when loading

By default, when button is loading, user cannot interact with button. This behavior can be disabled with isDisabledWhenLoading attribute.

Click to toggle

Block buttons

In case you need a button that completely fills the width of parent block you can use block attribute.

Styling

You can change button styles with css variables.
