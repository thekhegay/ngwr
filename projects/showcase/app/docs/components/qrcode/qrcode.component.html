QRCode

{{ description }}

How to use

Import WrQRCodeModule into a module where you want to use.

Usage example

QRCode component can be used as standalone wr-qrcode component

Size

You can customize QR code size with size attribute.

Error level

You can set QR code error level with level attribute.

With icon

QRCode component can be used with icon from url or with NGWR icons

Colored

You can customize QR code color with color attribute.

With custom background color

You can customize QR code background color with bgColor attribute.

API

Property Description Type Default
[value] set value string
[level] set error level for qrcode 'L' | 'M' | 'Q' | 'H' 'M
[padding] set padding number 10
[color] set color of qrcode string '#ffffff'
[size] set size of qrcide number 150
[icon] add icon to qrcode string | wrIconName
[iconSize] set size of icon number 42
