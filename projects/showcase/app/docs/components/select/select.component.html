Keyboard Navigation

Select component supports keyboard navigation for better user experience.

Key Description
↑ Navigate to previous option
↓ Navigate to next option
Enter Select highlighted option
Backspace Remove last selected item (in multiple mode)
Escape Close dropdown

API

Property Description Type Required Default
[placeholder] set placeholder of select string ""
[noItemsLabel] set "No Items" label text for list string ""
[disabled] set disabled state for select boolean false
[searchable] enable search functionality boolean false
[clearable] show clear button when value is selected boolean false
[multiple] enable multiple selection boolean false
[maxMultipleCount] maximum number of items that can be selected in multiple mode number undefined
