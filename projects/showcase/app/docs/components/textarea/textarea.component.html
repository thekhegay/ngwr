How to install

Import WrInputComponent into a component where you want to use.

Basic usage

Basic usage of input component

Prefix and suffix

You can add prefix or suffix attribute to add suffix and prefix to input.

Password input

If you need to display eye icons for password input, you can add passwordIcons attribute.

Disabled state

You can add disabled attribute to prevent interacting with the input.

Readonly state

You can add readonly attribute to prevent change of with the input.
