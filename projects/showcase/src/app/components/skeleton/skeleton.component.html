Colors

Skeleton component is available in different colors. Color can be set by color attribute.

API

Property Description Type Default
[color] set color of skeleton 'primary' | 'secondary' | 'success' | 'warning' | 'danger' | 'light' | 'medium' | 'dark' 'primary'
[animated] set animated state of skeleton boolean true'
